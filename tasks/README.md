# Tasks Snapshot

- `state-manager.task.xml` — Ghi/đọc state YAML, quản lý resume, liệt kê dự án.
- `fetch-and-save-content.task.xml` — Crawl URL, chuyển HTML → Markdown, lưu session/metadata.
- `analyze-content.task.xml` — Phân tích cấu trúc, tone, keyword density, xuất báo cáo song ngữ.
- `preliminary-market-research.task.xml` — Lấy dữ kiện thị trường và đối thủ hỗ trợ chiến lược SEO.
- `ideas-debate.task.xml` & `outline-debate.task.xml` — Dàn dựng tranh biện đa agent để chắt lọc ý tưởng/outline.
- `human-outline-approval-gate.task.xml` & `human-approval-gate.task.xml` — Cổng phê duyệt song song của con người trước khi tiến bước.
- `adaptive-writing.task.xml` — Viết lại bài dựa trên outline, persona, guideline và insight đã tổng hợp.
- `seo-optimization.task.xml` — Tối ưu tiêu đề, heading, từ khóa, meta, đề xuất schema.
- `quality-assurance-and-formatting.task.xml` — QA nội dung, chuẩn hóa định dạng, xuất YAML final.
